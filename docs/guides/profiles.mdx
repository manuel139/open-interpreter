---
title: Profiles
---

Profiles are a powerful way to customize your instance of Open Interpreter.

Profiles can be accessed with `interpreter --profiles`

Profiles can be set with `interpreter --profile name`

### Helpful fields

`interpreter.user_message_template` allows users to have their message wrapped in a template. This can be helpful steering a model to a desired behaviour without needing the user to add extra context to their message.
